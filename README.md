# SNNB-2019
Rotinas para execução de scripts para comparação entre descritores LBP e HOG usados em classificadores SVM, com base em imagens histológicas.

Comparar o desempenho de descritores de imagens LBP e HOG com classificador SVM.
LBP >>> Descritor de textura 
HOG >>> Descritor de contorno
SVM >>> Classificador Binário

Parametrização
1. Será utilizado estratégia de GridSearch para variação dos parâmetros no SVM.
2. Tanto no LBP quanto no HOG serão utilizado parâmetros constantes encontrados na literatura.
